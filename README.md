# Circuit simulation
FAIRmat programming assignment

# Clone the repo
```
git clone git@github.com:sprogl/fairmat.git assignment-toomani
cd assignment-toomani

```

# Run the code:
```
python -m venv .venv
source .venv/bin/activate
python circuit.py
```

or alternatively,

```
chmod +x circuit.py
./circuit.py
```
